Tips:

  1. Do not invert shapes
  2. Do not place nodes or parts of shapes inside of other shapes
  3. Do not drag objects out of the graph area; they will no longer be reachable if dropped
  4. *Note that only visibility edges on the convex hull are drawn when a concavity is present — no need to go into that :)
  5. When creating, type the number of vertices and select the create obstacle button
  6. When deleting, make sure the delete button is red and simply select obstacles
  7. When stepping through,
    • The red node is the current source for the rotational plane sweep
    • Sweep line is designated by a solid blue line
    • Visible vertices are highlighted light blue once visited by the source
    • Once all vertices have been visited by a source, purple visibility lines are drawn
